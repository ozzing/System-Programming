# 프로젝트 개요
	프로젝트 #1, #2에서 구현한 셸(shell)에 linking과 loading 기능을 추가하는 프로그램
	assemble명령을 통해서 생성된 object 파일을 link시켜 가상 메모리에 값을 저장하고,
	이를 실행하여 사용된 레지스터의 값을 출력한다.
	'sicsim>'에 셸 관련 명령어들을 입력하여, 명령어에 해당하는 기능을 수행 

# 실행 환경
	Linux (gcc)

# 프로그램 컴파일 방법
	~$ make

# 프로그램 컴파일 내용 삭제 방법
	~$ make clean		

# 프로그램 실행 방법
	먼저 프로그램을 컴파일한다.
	이후,
	~$ ./20191654.out
	을 통해 프로그램을 실행한다.
