# 프로젝트 개요
	프로젝트 #1에서 구현한 셸(shell)에 assemble 기능을 추가하는 프로그램
	SIC/XE의 assembly program source 파일을 입력 받아서 object파일을 생성
	어셈블리 과정 중 생성된 symbol table과 결과물인 object 파일을 보여줌
	'sicsim>'에 셸 관련 명령어들을 입력하여, 명령어에 해당하는 기능을 수행 

# 실행 환경
	Linux (gcc)

# 프로그램 컴파일 방법
	~$ make

# 프로그램 컴파일 내용 삭제 방법
	~$ make clean		

# 프로그램 실행 방법
	먼저 프로그램을 컴파일한다.
	이후,
	~$ ./20191654.out
	을 통해 프로그램을 실행한다.
